Laquu.JS

Laquu.JSはWebサイトの管理・制作をもっと楽に制作するために作られたjQueryプラグインパックです。 日頃の似たような作業を減らし、1ファイルで簡潔し、作業工程・精神への負担を減らせることができるプラグインです。

使い方

Laquu.JSの使い方は以下のとおりです。

<script type="text/javascript" src="/path/to/jquery.min.js"></script>
<script type="text/javascript" src="/path/to/laquu.js"></script>
<script type="text/javascript">
laquu(function(){
    // ここにLaquu.JSのコードを書きます
});
// もしくは
jQuery(function(){
    // ここにLaquu.JSのコードを書きます
});
</script>

Google Closure Compilerで最小化を行う

Laquu.JSはver1.2.0から、Google Closure Compilerを利用し、min構成を作っています。 build.sh（bash）を実行することで自動的に最小構成ファイルの全入りと、各プラグインの最小構成ファイルを書き出します。

$ cd Laquu.JS
$ chmod 0755 build.sh
$ ./build.sh
:
: いっぱいログが出ます
:
$ // 書き出し完了のメッセージが表示されれば完了です。
